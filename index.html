What is travel recce?

In a sentence, travel recce is an iOS travel app designed for the curation, and execution of the perfect day out.

The user is guided to build a trip, with a starting location of anywhere in the world, and to add destinations to their trip via an interactive map.

Once the user has curated their trip with a custom name and description, they are encouraged to add notes to each destination. For instance, 'get souvenirs from the museum'.

Finally, the user is able to view and play their trip on an interactive map, and choose to share the trip with their followers (coming soon).


Implemented

  • Trip creation/deletion

  • Adding/removing destinations to/from user defined trip

  • Location/destination mapping

  • Working search parameters for destinations - location search distance, establishment type

  • Dynamic map settings for different GPS views and live traffic conditions

  • Read existing google reviews for each destination

  • Visit a destination's website for more info

  • View a trip on the map, with destination lines between each waypoint

Yet to be implemented

  • Sharing of user trips to their profile

  • Allow commenting on shared trips via travel recce social

  • View nearby and trending destinations without creating a trip

  • Suggest destinations during trip curation

  • Extra destination search parameters - search by popularity, price, closest to user
